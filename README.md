# [SERVER STATIC] 

## Getting Started

In terminal:
```
$ git clone https://github.com/cleiton-tavares/server-static.git
$ cd server-static
$ sudo npm install [ LINUX / OSX ]
$ npm install [ WINDOWS ]
$ node server
```
